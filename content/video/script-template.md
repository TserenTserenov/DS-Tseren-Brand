# Шаблон сценария для телесуфлёра

## Два режима записи: когда суфлёр не нужен

Всё ниже - режим А (суфлёр, полный текст). Для историй, озарений, юмора и съёмки на ходу есть **режим Б - опорные точки**: суфлёр не используется вообще. Когда какой режим брать - `../../style/video.md`, раздел «Два режима записи».

### Режим Б - карточка опорных точек

1. В заметке на телефоне: первая строка - крючок дословно; дальше 3-5 опорных пунктов (по одной мысли на строку); последняя строка - финал дословно.
2. Прочитать карточку дважды вслух - и убрать из поля зрения.
3. Снять один дубль целиком. Запнулся - продолжай: переснимать стоит потерянную мысль, не запинку.
4. В конце сессии перезаписать крючок: голос уже разогрет, первые секунды выйдут лучше.

### Съёмка на ходу (прогулка / машина)

- **Звук решает:** петличка или гарнитура; без них на улице не снимать.
- Телефон в вытянутой руке, объектив чуть выше уровня глаз, держать горизонт; шаг медленнее обычного.
- Машина - только припаркованная: свет из лобового стекла, телефон на держателе.
- Прогулочный ролик может быть короче студийного: 30-60 сек.

## Инструмент: выбор под твою установку

**Важно:** Parrot показывает текст, но не снимает видео. Нужно выбрать схему под свои устройства.

### Схема А — два устройства (рекомендую)

Parrot на старом айфоне или айпаде показывает текст рядом с камерой.
Снимаешь основным айфоном на заднюю камеру.
- Качество видео лучше (задняя камера)
- Взгляд более естественный (смотришь на текст рядом, а не на экран)
- Parrot: App Store → "Parrot Teleprompter"

### Схема Б — один айфон

Нужно приложение, которое совмещает суфлёр и запись в одном экране.
Текст скроллится прямо поверх изображения с камеры.
- **BigVu** — бесплатный план, суфлёр + запись в одном, есть App Store
- Минус: фронтальная камера, качество ниже

### Workflow (для обеих схем)

1. Скопировать сценарий из карточки темы (`video/ideas/`)
2. Вставить в Parrot (схема А) или BigVu (схема Б)
3. Настроить: шрифт крупный, скорость под себя, фон тёмный
4. Снять

## Откуда берётся сценарий

Каждая карточка темы (`video-theme`) уже содержит раздел «Сценарий (под суфлёр)» — текст разбит на короткие строки (4–8 слов), готов к чтению. Копировать оттуда.

**Базовая длительность по темпу речи:**
- 100 слов ≈ 40–45 сек (нормальный темп)
- 120 слов ≈ 50–55 сек
- 150 слов ≈ 60–65 сек
- 180 слов ≈ 70–75 сек

Подсчёт слов: `wc -w` в терминале или любой онлайн-счётчик.

## Адаптации по каналам

### TikTok / YouTube Shorts (≤60 сек)

**Цель:** первые 2 секунды = крючок, который останавливает скролл.  
**Темп:** быстрый, без пауз внутри предложений.  
**CTA:** одно слово в конце: «подписывайся», «сохрани», «делись».

**Правила:**
- Первая строка — провокация или вопрос (не «сегодня поговорим о»)
- Убрать вводные слова: «итак», «знаете ли вы», «давайте разберёмся»
- Максимум 140 слов для TikTok, 150 для Shorts
- Финал: 1 предложение, без длинных объяснений

```
[КРЮЧОК — 2 сек]
Первое предложение — удар.

[ТЕЛО — 40-50 сек]
... основной сценарий ...

[ФИНАЛ — 5-8 сек]
Одна ёмкая фраза.
```

---

### Telegram видео / Reels для Facebook (60–90 сек)

**Цель:** полная мысль, есть место дышать.  
**Темп:** чуть медленнее, можно паузы между блоками.  
**CTA:** может быть длиннее — ссылка на пост, приглашение в обсуждение.

**Правила:**
- Крючок всё равно нужен в первые 3 секунды
- Можно добавить 1 пример или аналогию (чего нет в Shorts)
- Финал: 2–3 предложения, включая CTA
- Максимум 200 слов

```
[КРЮЧОК — 3 сек]
Первое предложение.

[ТЕЛО — 50-70 сек]
... основной сценарий + 1 пример ...

[ФИНАЛ — 10-15 сек]
Вывод.
CTA.
```

## Практический пример: «ТВС — три типа работы»

**Исходник** (из `video/ideas/2026-06-20-tws-tri-tipa-raboty.md`):

```
У вас три типа работы.
Не один.

Есть текущее — то, что делаете по плану каждый день.
Есть срочное — когда что-то грозит остановить весь процесс.
И есть важное — когда строите что-то новое, лучшее.

Большинство людей смешивают их в одну кучу.
И теряются.

Текущее — это плановый выпуск.
Срочное — это пожар, который надо потушить прямо сейчас.
Важное — это модернизация, для которой нужно специально выделять время.

Если не разделять — всё время тушишь пожары,
которых, может быть, вовсе нет.

Попробуйте сегодня:
посмотрите на свой список задач
и спросите — это текущее, срочное или важное?

Результат удивит.
```

**Для TikTok/Shorts** (≤60 сек, ~140 слов — подходит без правок):
Этот сценарий уже в нужном формате. Вставить в Parrot как есть.

**Для Telegram/Facebook** (добавить пример):
После «И есть важное — когда строите что-то новое, лучшее.» добавить:

```
Например, учиться новому методу —
это важное, не текущее и не срочное.
Если не выделить время специально — оно никогда не случится.
```

## Физическая установка

**Что нужно:** мини-штатив (1 000-1 500 ₽ на Wildberries, высота ~25 см, с зажимом для телефона). Без штатива снимать с суфлёром на том же экране не получится — нужны свободные руки.

### Вариант А - сидя за столом (проще)
Мини-штатив на столе. Телефон на уровне глаз или чуть выше — смотришь на экран слегка вверх, лицо выглядит увереннее. Расстояние — 50–70 см от лица. Садишься прямо, плечи чуть назад.

### Вариант Б - стоя (энергичнее)
Штатив на высокой полке или стопке книг — телефон на уровне лица. Снимаешь стоя. Подходит для более динамичных тем.

**Трюк с глазами:** объектив фронтальной камеры — маленький кружок в верхней части экрана. Смотри туда, а текст читай периферийно. Чем крупнее шрифт и чем ближе текст к верхнему краю экрана — тем меньше взгляд уходит вниз в кадре.

## Parrot: точные настройки

### Шаг 1 — создать скрипт
- Открыть Parrot → кнопка «+» → «New Script»
- Вставить текст из карточки (раздел «Сценарий под суфлёр»)
- Название — дата + slug: `20jun-tws`

### Шаг 2 — настройки (делаются один раз, сохраняются)

| Параметр | Где найти | Значение |
|----------|-----------|---------|
| Font Size | кнопка «Aa» в редакторе | 60–70pt; увеличь, пока текст читается без прищура с 60 см |
| Background | цветовой круг | чёрный фон + белый текст |
| Speed | ползунок 1–10 | начни с 6/10; убегает — 7–8, не успеваешь — 4–5 |
| Countdown | шестерёнка → Countdown | 3 секунды |
| Mirror | шестерёнка → Mirror | выключить |
| Camera | значок камеры | Front (фронтальная) |

### Шаг 3 — калибровка скорости (5 минут, один раз)
1. Нажми Play, не снимая — просто читай вслух с таймером
2. Текст убегает вперёд → скорость вниз
3. Ждёшь текст → скорость вверх
4. Идеально — ты чуть-чуть опережаешь текст, не ждёшь его

### Шаг 4 — снять
- Play → 3 сек обратного отсчёта → читай
- Пауза — тап по экрану
- Второй дубль — свайп назад

**Первый раз перед съёмкой:** прочитай вслух с секундомером без суфлёра — замерь темп. Потом выстави скорость в Parrot под свой темп и прогони один раз. Только потом снимай.
