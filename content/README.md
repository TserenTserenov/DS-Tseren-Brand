# Контент-производство

> Все черновики и готовые материалы для публикации. Организовано по формату - редактор идёт в `posts/`, видеопродакшн - в `video/`. Тема - атрибут внутри файла (поле О1-О6 во frontmatter), не папка.
>
> **Контент-система:** определения осей (О, И, форматы, места) - в [`../08-content-themes.md`](../08-content-themes.md).

## Форматы

| Папка | Что | Статусы |
|-------|-----|---------|
| [posts/](posts/) | Текстовые черновики для постов, тредов, статей | draft / ready / published |
| [video/](video/) | Сценарии коротких видео | 💡 идея · 🎬 записано · 📤 опубликовано |
| [templates/](templates/) | Шаблоны структур по типам текстов | — |

## Как добавить материал

1. Определи формат (пост, видео, статья).
2. Положи в нужную папку (`posts/` или `video/ideas/`).
3. Добавь frontmatter с полями `О` (область О1-О6), `И` (ипостась И1-И6), `status`.
4. Добавь строку в индекс своей папки (README.md папки).
